# Keys

## Shortkeys

Ctrl + Cmd + Q Lock screen
Ctrl + Cmd + F Toggle full screen
Ctrl + Cmd + Space Emojis etc,

Shift + Cmd + Q Log out
Shift + Cmd + 5 Screenshot
Shift + Cmd + . Show dot files
Shift + Cmd + H Open the home folder
Shift + Cmd + G Open a Go to Folder window

Shift + Ctrl + ^ Toggle terminal

Shift + Opt + Mouse  Column selection
Shift + Opt + F12 Find All References

Cmd + K Connect to server
Cmd + F Find
Cmd + Space Spotlight

Ctrl + Up Arrow Mission Control
Ctrl + Left Arrow Space left
Ctrl + Right Arrow Space right
Ctrl + - Previous location

F12 Go to Definition
